YoloBox Mini: Експертна База Знань

All-in-One Encoder, Switcher & Monitor Guide

1. Залізо та Кнопки (Physical Layout)

Фізичні елементи

  • Кнопка живлення: Зверху праворуч. Коротке натискання — сон, довге — увімкнення/вимкнення.
  • SIM-слот: Зліва. Підтримує 4G LTE.
  • SD-слот: Знизу. Для запису та медіафайлів.
  • Різьба 1/4": Для кріплення на камеру або штатив.

Порти підключення

  • HDMI In: Основна камера.
  • USB-A: Вебкамера або USB-мікрофон.
  • Audio In (3.5mm): Mic/Line рівень (перемикається в меню).

2. Налаштування Мережі (Network)

Пріоритети підключення

  1. Ethernet: Завжди №1 для стабільності.
  2. 4G LTE: Потребує Nano-SIM (без PIN-коду).
  3. Wi-Fi: Тільки за умови стабільного сигналу роутера.

Bonding (Об'єднання)

Об'єднання Ethernet + 4G + Wi-Fi працює через платну підписку YoloLiv Cloud.
Шлях: Account → Network Settings.

3. Сервер та Стрімінг (RTMP)

Запуск трансляції

  1. Створіть "Live Stream", натиснувши "+".
  2. Додайте платформу через Add або виберіть Custom RTMP.
  3. Введіть Server URL та Stream Key.
  4. Натисніть "GO LIVE" для старту.

4. Налаштування Звуку (Audio Mixer)

Лайфхак: Використовуйте режим AFV (Audio Follow Video) тільки для 2+ джерел. Для однієї камери ставте звук "On".
Функція Призначення Рекомендація
Main Volume Майстер-рівень Піки в межах -6..-3 дБ
Audio Delay Синхронізація звуку 40-100 мс, якщо звук випереджає картинку

5. Запис на SD-карту

Критично: Використовуйте тільки карти U3 (V30). Звичайні карти призводять до зависання пристрою.
