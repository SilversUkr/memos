YoloBox Mini: Швидкий Старт

📞 Відповідальний за трансляції (Артем): +380 99 432 8225

Пароль SIM-карти: 8012 або відсутній

1. Підготовка та Перевірка

Monitor Mode
  • Натисніть кнопку Monitor Mode на головному екрані.
  • Перевірте якість відео та звуку. Якщо все ОК — поверніться в головне меню (кнопка Back).
  • Для Wi-Fi: Settings → Network Settings → Підключіться до мережі.

2. Створення Події

+
Create Live Stream
  • Натисніть великий помаранчевий +.
  • Оберіть Create Live Stream.
  • Введіть будь-яку назву та натисніть Create (інші дані не потрібні).

3. Налаштування Звуку (КРИТИЧНО!)

Зайдіть у створений стрім. Знайдіть іконку динаміка в бічному меню.

HDMI Audio:
ON
УВАГА: Обов'язково увімкніть звук для HDMI потоку. Він вимикається автоматично після кожного завершення трансляції! Слідкуйте за шкалою рівня звуку.

4. Вибір Каналу та Старт

Connectors: СтопКорчик
  • Зайдіть у Connectors.
  • Оберіть СтопКорчик (або інший за вказівкою Артема).
  • Натисніть GO LIVE для початку ефіру.
